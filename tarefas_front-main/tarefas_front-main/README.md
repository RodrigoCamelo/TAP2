# Tarefas Frontend

Frontend da aplicação carros unifacisa.

## Tecnologias utilizadas

- ReactJS
- JSX
- CSS
- AXIOS

## Pré-requisitos

Você deve ter o mongodb e o backend em execução.

## Executando a aplicação

Você deve:

1. baixar o repositório;
2. instalar as dependências;
3. iniciar o frontend;
